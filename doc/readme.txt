引擎说明
1.来源于cocos2d-html5 可以认为是简化版cocos引擎
2.优先解决渲染层次问题，程序员更加关注游戏内容逻辑  已经实现
3.提供事件注册侦听机制  实现最基本touch 但没有处理多层优先级问题
4.提供定时器方法       暂不处理（这个功能是用来处理定间隔回调方法 可以使用setInterval()）
5.提供常用tween动画    move rotate 动画序列 循环 回调已经实现
6.提供帧动画           已经实现
7.增加对plist文件纹理图支持 已经实现

引擎的大循环是setInterval定时器，其时间间隔由帧频确定  其中 重绘 动画效果 都在此循环中处理 本质是根据dt参数进行回调分发
事件侦听机制 采取canvas的touch事件或者mouse事件捕获，之后将事件封装冒泡至eventManager
引擎的坐标系统采用比较熟悉的笛卡尔坐标系 对 canvas api中的坐标系做了转换 同时引入绘图锚点的概念
对于需要原生绘图的地方，可以继承sprite类然后重写draw方法，draw方法会传入ctx对象可以直接使用ctx绘图
但应该注意save和restore防止对ctx的更改影响到其他精灵类的绘制


以上模块的代码 仿照了cocos2d-html5游戏引擎源码书写，其中部分代码复用，旨在解决最基本的游戏需求，同时降低引擎文件大小
